Detailed Description

Describes an optional interface that a pooled MonoBehaviour can implement in order to receive callbacks on borrow and revert operations performed on its instances

An instance of a GameObject can contain multiple components implementing this interface. In this case the callbacks will be called on ALL the components of the object (But not their children!)
